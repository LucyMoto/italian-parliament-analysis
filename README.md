# Detecting Factional Fractures in the Italian Parliament

## Problem Statement

Italian parliamentary parties are often portrayed as monolithic blocs, but roll-call votes tell a different story. This project quantifies internal party cohesion over time and detects emerging factional splits using voting data from the Camera dei Deputati.

## Scope

- **Legislature:** XIX (October 2022–present)
- **Chamber:** Camera dei Deputati
- **Data source:** Roll-call votes scraped from [documenti.camera.it](https://documenti.camera.it/apps/votazioni/votazionitutte/FormVotazioni.Asp?Legislatura=19)

## Analysis Pipeline

1. **Data ingestion** — Scrape roll-call vote detail pages for individual deputy votes
2. **Cohesion index** — Rice index per party per month/quarter
3. **Legislator similarity clustering** — Deputy-by-deputy voting similarity, dimensionality reduction (UMAP/PCA), clustering (DBSCAN/hierarchical)
4. **Defection detection** — Flag deputies diverging from party line; identify which policy domains drive splits
5. **Visualization** — Interactive dashboard (Streamlit) with cohesion trends, cluster maps, top defectors

## Setup

```bash
# Clone
git clone https://github.com/LucyMoto/italian-parliament-analysis.git
cd italian-parliament-analysis

# Create environment
python -m venv venv
source venv/bin/activate  # macOS/Linux
# venv\Scripts\activate   # Windows

# Install dependencies
pip install -r requirements.txt
```

## Usage

```bash
# 1. Scrape sample data
python src/scraper.py

# 2. Clean and process
python src/clean.py

# 3. Run analysis
python src/analysis.py

# 4. Launch dashboard
streamlit run src/dashboard.py
```

## Data Notes

- Raw data is scraped from public parliamentary records and can be regenerated by running the scraper
- Each roll-call vote page contains: deputy name, parliamentary group, individual vote (Favorevole/Contrario/Astensione/Non ha partecipato/In missione)
- The scraper includes polite delays (1s) between requests
